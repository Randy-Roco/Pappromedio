# PromedioPApp Web

Versión web estática, lista para desplegar en Vercel.

## Archivos
- index.html
- styles.css
- app.js
- vercel.json

## Qué hace
- Carga múltiples archivos TXT
- Detecta y normaliza descriptores PA
- Permite aliases manuales
- Promedia X/Y/Z por descriptor
- Exporta TXT para Civil 3D y ERDAS
- Exporta Excel con hojas de Promedios y Detalle

## Cómo integrarlo en tu repo
Copia estos archivos a la raíz del repositorio o a una carpeta web dedicada.

## Deploy en Vercel
1. Haz push al repo en GitHub
2. Importa el repo en Vercel
3. Framework Preset: Other
4. Root Directory: / (o la carpeta donde dejes estos archivos)
5. Deploy
